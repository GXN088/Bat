# Bat
Класс Bat (летучая мышь) унаследован от класса Animal. Все логично, вот только при вызове метода move() у объекта класса Bat выведется в консоли "Я бегу!".
Зачем бежать, если ты умеешь летать?
Переопредели метод move() для класса Bat, чтобы он выводил в консоли "Я лечу!".
Метод main() в тестировании участия не принимает.


Псевдокод:

```
// Основной класс программы
CLASS Solution
    METHOD main(args)
        // Создаем объект класса Bat
        bat = NEW Bat()
        
        // Вызываем метод move() у объекта bat
        bat.move()
    END METHOD
END CLASS

// Базовый класс Animal
CLASS Animal
    METHOD move()
        PRINT "Я бегу!"
    END METHOD
END CLASS

// Производный класс Bat, наследуется от Animal
CLASS Bat EXTENDS Animal
    // Переопределение метода move()
    METHOD move()
        PRINT "Я лечу!"
    END METHOD
END CLASS
```

Объяснение логики:

1. Наследование: Класс Bat наследуется от класса Animal
2. Полиморфизм: Класс Bat переопределяет метод move() родительского класса
3. Выполнение программы:
   · Создается объект класса Bat
   · Вызывается метод move()
   · Поскольку метод переопределен в классе Bat, выполняется версия "Я лечу!" вместо родительской "Я бегу!"

Результат выполнения:

```
Я лечу!
```

Это демонстрация принципа полиморфизма в ООП, где производный класс изменяет поведение унаследованного метода.
